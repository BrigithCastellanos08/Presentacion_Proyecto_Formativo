COFFEE SOFT — Presentación del proyecto
========================================

CÓMO ABRIRLA
1. Descomprime la carpeta.
2. Haz doble clic en "index.html" (se abre en cualquier navegador). No necesita internet, excepto para cargar las fuentes de Google Fonts la primera vez.

NAVEGACIÓN
- Flechas ← → del teclado, barra espaciadora, o los botones circulares a los lados.
- Los puntos de la derecha saltan directo a cualquier diapositiva.
- En móvil/tablet: desliza el dedo hacia la izquierda o derecha.

CAMBIAR LAS IMÁGENES
Todas las imágenes están en la carpeta /img con nombres descriptivos:
  logo-coffeesoft.png        -> logo del proyecto (portada, cierre y manual de marca)
  diagrama-conceptual.png    -> modelo conceptual
  diagrama-logico.png        -> modelo lógico MySQL
  diagrama-clases.png        -> diagrama de clases
  diagrama-casos-uso.png     -> diagrama de casos de uso
  diagrama-mer.png           -> modelo entidad-relación
  arquitectura-sistema.png   -> arquitectura del sistema

Para reemplazar una imagen, solo sustituye el archivo por otro CON EL MISMO NOMBRE
dentro de /img. No es necesario tocar el HTML ni el CSS.
Todas las imágenes se amplían al hacer clic sobre ellas (lightbox).

EDITAR TEXTOS
Abre "index.html" con cualquier editor de texto (VS Code, Bloc de notas, etc.)
y busca el texto que quieras cambiar; está todo en español y organizado
por diapositiva con comentarios "<!-- N. NOMBRE -->".

Los campos marcados entre corchetes, por ejemplo [Escribe tu nombre aquí],
están pensados para que los reemplaces con tus datos antes de exponer.

ARCHIVOS
  index.html   -> estructura y contenido de las 17 diapositivas
  style.css    -> diseño visual (colores, tipografía, layout, animaciones)
  script.js    -> navegación entre diapositivas y lightbox de imágenes
  img/         -> todas las imágenes de la presentación
